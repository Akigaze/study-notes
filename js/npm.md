## 简写列表
* 安装模块时 --save 是默认选项，可以省略不写
* i 是 install 的简写
* -y 是 --yes 的简写
* -g 是 --global 或 -global 的简写，但使用全称时最好使用 --global
* -D 是 --save-dev 的简写

## 默认初始化
> npm i -y  


## 查看所有安装的全局模块
> npm list --depth=0 -global

## 卸载已安装的全局模块
> npm uninstall -g <package>

## npm全局模块的安装目录
> C://Users//username//AppData//Roaming//npm

## 更新npm
> npm install npm@latest -g

## 更新项目中的模块
> npm update <package>

## 降低模块版本
> npm install <package>@version
