# Spring Basic
## What is Spring?
1. 开源
2. 简化企业级应用
3. IOC和AOP的容器框架

## Description of Spring
1. 轻量级，非入侵
2. 依赖注入(DI: Dependency Injection)
3. 面向切面编程(AOP: Aspect Oriented Programming)
4. 容器，管理对象的生命周期
5. 框架
6. 一站式
