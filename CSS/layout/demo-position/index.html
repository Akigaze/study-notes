If the position is not static(default), we say that element is positioned

position relative

使用position: relative 的元素，文档流会保留该元素原有的位置不被其他文档流中的元素占领，同时 top，left 等属性设置的是相对于原有位置的偏移

position fixed

使用position: fxed 的元素会失去在文档流中的位置而被其他文档流中的元素占领，同时 top，left 等属性设置的是相对于浏览器窗口的位置偏移
margin: auto 在左右方向要生效只有同时设置了 left 和 right 才行.
当没有设置 left 或 right 时，元素会紧靠父元素的左边；当没有设置 top 或 bottom 时，元素会保持原有所在高度

position absolute

position: absolute 的定位方式与 fixed 有些相似，它是以最近的一个 positioned 的父元素或 body 为基准进行偏移的
