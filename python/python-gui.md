## 海龟画图 turtle

`turtle` 模块提供了一系列画图的方法，它的工作方式是打开窗口按照代码指令的顺序在窗口中绘图

```python
from turtle import *
```

`turtle` 模块的实现某一功能的方法通常有多个，也可以认为是方法有它相应的简称，比如 `pensize()` 方法用于修改笔的粗细，而 `width()` 方法也是一样的功能
