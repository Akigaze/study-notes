## 贝叶斯算法
贝叶斯定理：要求特征之间相互独立  
    P(B|A) = P(A|B)P(B)/P(A)  
1. 要求特征之间相互独立，基于概率密度的算法
2. 适合做文本分类，但无法理解特征之间的隐含关系

## Python中的贝叶斯
`sklearn.naive_bayes`  
1. GaussianNB：连续的特征值
2. MultinomialNB：适合处理文本
3. BernoulliNB：0-1分布
